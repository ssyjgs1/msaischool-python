mmdetection conda 만들기

https://visualstudio.microsoft.com/visual-cpp-build-tools/ (뭐 안 되면 visual studio installer로 c++빌드툴 설치)

conda create -n mmdetection python=3.8
pip install torch==1.11.0+cu113 torchvision==0.12.0+cu113 torchaudio==0.11.0 --extra-index-url https://download.pytorch.org/whl/cu113
pip install mmcv-full==1.6.2 -f https://download.openmmlab.com/mmcv/dist/cu113/torch1.11/index.html

git clone https://github.com/open-mmlab/mmdetection (mmdetection 폴더에서)

pip install -r requirements/build.txt
pip install -v -e.
pip install mmdet
